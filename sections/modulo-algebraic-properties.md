# Algebraic Props

## Negative

$$   A \text{ \% } B = A \pmod{B} = (B-A) \pmod{B} \tag*{ if B > A}$$

eg.

$$   3 \text{ \% } 7 = 3 \pmod{7} = -4 \pmod{7}    $$

## Distributive

$$   (A + B) \text{ \% } C = A \text{ \% }C + B \text{ \% } C  $$

eg.

$$   (5 + 7) \text{ \% } 3 = 2 + 1 \pmod{3} = -1 + 1 \pmod{3} = 12 \text{ \% }3 = 0\pmod{3}    $$
